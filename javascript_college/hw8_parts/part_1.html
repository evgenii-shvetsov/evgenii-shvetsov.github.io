Rest API: RapidAPI (Yahoo Finance API)

The RapidApi provides you with access to query a database of the financial data: Stocks, Quotes, Charts, and Finance Summaries.

By using this API you will be able to get a JSON response file.

Unfortunately, the Yahoo Finance API on RapidAPI is not free.
There is one Basic Plan, it will allow you to get 500 API calls for free, then you have to pay $0.002 per additional API call.
There are several Pricing Plans, the most expensive one for $200/month will offer you unlimited API requests.
